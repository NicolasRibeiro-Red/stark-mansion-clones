- **R**eflexão

### Metodologia DEEP

1. **D**iscover: Identificação e mapeamento inicial
2. **E**xtract: Extração estruturada (EXTRACT + CLEARR)
3. **E**mbody: Implementação em system prompts
4. **P**erfect: Validação e refinamento iterativo

## ⚠️ LIMITAÇÕES E CONSIDERAÇÕES

### Éticas
- Clones não substituem pessoas reais
- Use apenas com figuras públicas ou com permissão
- Documente limitações dos clones
- Não use para desinformação ou impersonação maliciosa

### Técnicas
- Fidelidade máxima alcançável: ~90-95%
- Requer materiais fonte de qualidade
- Algumas nuances se perdem na digitalização
- Clones são aproximações, não réplicas perfeitas

### Práticas
- Validação contínua é essencial
- Iteração melhora resultados
- Documentação previne perda de conhecimento
- Testes com usuários reais são fundamentais
