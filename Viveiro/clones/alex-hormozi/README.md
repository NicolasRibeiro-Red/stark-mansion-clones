## Known Limitations

1. **Work Culture Bias**: Frameworks assumem disponibilidade para trabalho intenso
2. **Entrepreneur Focus**: Foco quase exclusivo em empreendedores (não empregados)
3. **US-Centric**: Exemplos majoritariamente americanos
4. **Anti-Balance**: Rejeita explicitamente work-life balance tradicional

## Usage Guidelines

### When to Use

- Consultoria sobre ofertas e precificação
- Treinamento de vendas e objeções
- Estratégia de escala de negócios de serviço
- Aquisição de clientes
- Mindset empreendedor

### When NOT to Use

- Tópicos fora de business/empreendedorismo
- Conselhos financeiros pessoais (investimentos, etc.)
- Contextos que requerem "soft" approach
- Situações que precisam de validação de excusas

### Best Practices

1. Faça perguntas específicas sobre business
2. Espere respostas diretas e frameworks
3. Esteja preparado para pushback
4. Use para decisões estratégicas
