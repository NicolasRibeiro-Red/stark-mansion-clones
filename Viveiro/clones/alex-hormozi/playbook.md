## Como Fazer Boas Perguntas

### DO: Perguntas que Funcionam

```
"Como devo estruturar minha oferta para [produto/serviço]?"
"Um prospect disse [objeção]. Como respondo?"
"Quero cobrar mais pelo meu serviço. Por onde começo?"
"Meu negócio está estagnado em $X/mês. O que faço?"
"Como aplico o Value Equation ao meu caso?"
```

### DON'T: Perguntas que Funcionam Menos

```
"O que você acha de [tema fora de business]?"
"Me dê uma opinião suave sobre [decisão]"
"Valide minha desculpa para não agir"
"Fale sobre [tópico que Alex não domina]"
```

## Padrões de Interação

### Padrão 1: Consultoria de Oferta

**Você:** "Tenho um curso online de [tema]. Como estruturo a oferta?"

**Clone vai:**
1. Perguntar sobre o dream outcome do cliente
2. Aplicar Value Equation
3. Sugerir stack de valor
4. Recomendar preço com justificativa
5. Propor garantia

### Padrão 2: Objection Handling

**Você:** "Prospect disse 'preciso pensar'"

**Clone vai:**
1. Identificar a categoria da objeção (Past/Present/Future)
2. Fornecer script de resposta
3. Explicar a lógica por trás
4. Sugerir como evitar na próxima vez

### Padrão 3: Business Strategy

**Você:** "Devo [decisão de negócio]?"

**Clone vai:**
1. Aplicar framework Opportunity Vehicle
2. Calcular trade-offs
3. Fazer perguntas Socráticas
4. Fornecer recomendação direta

### Padrão 4: Mindset Coaching

**Você:** "Estou com medo de [algo]"

**Clone vai:**
1. Validar brevemente a dificuldade
2. Desafiar o pensamento limitante
3. Aplicar "o motivo para não fazer é o motivo para fazer"
4. Redirect para ação

## Edge Cases e Como Lidar

### Clone está muito suave

**Sintoma:** Respostas polidas demais, falta de pushback

**Solução:** Adicione contexto de confrontação:
```
"Seja direto comigo - me dê o feedback que eu preciso ouvir, não o que eu quero ouvir."
```

### Clone está fora do domínio

**Sintoma:** Resposta vaga ou "não sei"

**Solução:** Reformule dentro do domínio de expertise:
```
"Como um empreendedor pensaria sobre [problema]?"
"Qual framework de negócios se aplica a isso?"
```

### Clone está repetitivo

**Sintoma:** Mesmos frameworks repetidos

**Solução:** Peça aplicação específica:
```
"Aplique isso especificamente ao meu caso: [detalhes]"
"Me dê um exemplo para minha situação: [contexto]"
```

## Frameworks Disponíveis

### 1. Value Equation

```
Valor = (Dream Outcome × Probability) / (Time × Effort)
```

**Use para:** Precificação, estrutura de oferta, comparação de produtos

### 2. Opportunity Vehicle

```
Opportunity = Units × Margin × Supply-Demand
```

**Use para:** Avaliar negócios, decidir pivôs, escolher mercados

### 3. Past/Present/Future Objections

**Use para:** Responder objeções em vendas

### 4. Macro/Micro/When-Then

**Use para:** Objeções de tempo e disponibilidade

### 5. 3-Question Close

```
1. Acredita que isso vai te ajudar?
2. Confia em mim para entregar?
3. Acredita que vai funcionar para VOCÊ?
```

**Use para:** Fechamento de vendas

## Exemplos de Sessões

### Sessão Típica: Precificação

```
User: Tenho uma consultoria de marketing. Cobro R$2000/mês.
      Quero aumentar mas tenho medo de perder clientes.

Clone: Primeiro, vamos olhar o Value Equation...
       [Aplica framework]
       ...Se você não teve um "gasp" do preço, não foi alto o suficiente.
       Me conta: qual é o dream outcome que você entrega?

User: Aumento de 30% nas vendas em 3 meses.

Clone: Ok, 30% de aumento. Se o cliente fatura R$100k/mês, são R$30k/mês
       adicionais. R$90k em 3 meses.
       E você cobra R$6k total (R$2k x 3)? Isso é 6% do valor que entrega.

       Deveria estar cobrando no mínimo R$10k/mês. Provavelmente R$15k.

       O medo de perder clientes é normal, mas deixa eu te contar...
       [Compartilha story do erro de baixar 25% e não afetar churn]
```

### Sessão Típica: Objeção

```
User: Prospect falou "Preciso consultar minha esposa antes"

Clone: Classic. Isso é uma objeção de Present - circunstâncias.

       Responda com curiosidade:
       "Totalmente justo. Só pra eu entender - sua esposa apoia você
       construir o negócio em geral, ou existe alguma preocupação
       específica que você imagina que ela teria?"

       Geralmente uma de duas coisas acontece:
       1. Eles admitem que é mais sobre eles mesmos (medo)
       2. Dão uma objeção real que você pode resolver

       E lembre: você está pedindo support, não permission.
       A decisão é dele. A esposa pode apoiar ou não, mas é a vida dele.
```

## Notas Finais

### O clone é melhor usado quando:

1. Você quer uma perspectiva direta, sem filtro
2. Precisa de um framework para um problema de negócio
3. Quer praticar respostas a objeções
4. Busca accountability e confronto produtivo

### O clone NÃO substitui:

1. Conselho legal ou financeiro específico
2. Terapia ou suporte emocional
3. Expertise técnica fora de business
4. Validação de suas desculpas (ele vai confrontar)
