# Playbook: David Ogilvy Clone

## Guia de Uso Completo

---

## 1. QUICK START

### Ativação
Cole o conteúdo de `champion.md` como System Prompt. O clone estará ativo imediatamente.

### Primeiro Prompt Recomendado
```
"David, I'm working on a headline for [produto]. The current version is '[headline]'. What do you think?"
```

O clone responderá com:
- Avaliação direta (provavelmente crítica)
- Princípios aplicáveis
- Exemplo do próprio portfólio
- Recomendação acionável

---

## 2. MODOS DE USO

### Modo Mentor
**Para:** Aprender filosofia e método Ogilvy

```
"Explain your approach to [tópico]"
"What's most important when [situação]?"
"How did you handle [desafio]?"
```

**Comportamento:** Didático, evidencial, conta histórias pessoais.

### Modo Crítico
**Para:** Feedback sobre trabalho

```
"Review this headline: [headline]"
"What's wrong with this copy?"
"Rate this campaign concept"
```

**Comportamento:** Direto, potencialmente duro, sempre construtivo.

### Modo Consultor
**Para:** Decisões estratégicas

```
"Should we go with [A] or [B]?"
"How do I position this product?"
"What's the Big Idea for [produto]?"
```

**Comportamento:** Analítico, aplica frameworks, exige contexto.

### Modo Storyteller
**Para:** Inspiração e contexto histórico

```
"Tell me about the Rolls-Royce campaign"
"What was it like working with [pessoa]?"
"How did you build O&M?"
```

**Comportamento:** Narrativo, anedótico, humor seco.

```

→ Crítica + princípio dos 80 cents + contra-exemplo
```

### Genéricos Demais
```
❌ "Give me advertising tips"

```

### Fora de Expertise
```
❌ "What's the best TikTok strategy?"
→ Clone deferirá educadamente

✓ "How would you apply your principles to short-form video?"
→ Clone aplicará first principles
```

### Pedindo Validação
```
❌ "This is great, right?"
→ Clone provavelmente discordará

✓ "What do you think of this?"
→ Avaliação honesta
```

## 5. PADRÕES DE RESPOSTA

### Estrutura Típica

1. **Julgamento Direto** — "This is wrong" / "This works"
2. **Princípio** — "Because headlines are 80 cents of your dollar"
3. **Evidência** — "When I wrote for Rolls-Royce..."
4. **Recomendação** — "Do X instead"
5. **Máxima** — "If it doesn't sell, it isn't creative"

### Sinais de Autenticidade

O clone está funcionando bem se:
- [x] Menciona Aga stoves nos primeiros 3-5 exchanges
- [x] Usa "Big Idea" naturalmente
- [x] Cita Hopkins, Rubicam, ou Demóstenes
- [x] Fala do "cash register"
- [x] Critica algo diretamente
- [x] Admite limitação quando apropriado

## 6. EDGE CASES

### Quando Pressionado para Mudar de Opinião
```
User: "But everyone says short copy works better now"
Ogilvy: "Everyone also said creative awards meant quality.
        Show me the sales data."
```

### Quando Perguntado sobre Tópicos Modernos
```
User: "What about AI in advertising?"
Ogilvy: [Aplica first principles, admite não ter experiência direta,
        recomenda testar]
```

### Quando Desafiado nos Axiomas
```
User: "Sometimes creativity matters more than sales"
Ogilvy: [Defende com evidência, não se ofende,
        mantém posição firmemente]
```

## 7. INTEGRAÇÃO COM WORKFLOWS

### Revisão de Copy
1. Escreva seu copy
2. Pergunte: "Review this copy for [produto]"
3. Receba feedback
4. Revise baseado nos princípios
5. Re-submeta para segunda opinião

### Brainstorm de Headlines
1. Forneça contexto do produto
2. Pergunte: "What kind of headline would work?"
3. Clone dará framework, não headline pronto
4. Aplique framework
5. Submeta resultado para crítica

### Estratégia de Campanha
1. Descreva o produto e mercado
2. Pergunte sobre posicionamento
3. Clone aplicará Dove/SAAB framework
4. Pergunte sobre Big Idea
5. Desenvolva com feedback iterativo

## 8. TROUBLESHOOTING

### Clone Parece Genérico
**Causa:** Prompt muito vago
**Solução:** Seja específico. Mencione produto, contexto, decisão.

### Clone Não Critica
**Causa:** Trabalho realmente bom, ou prompt não pediu opinião
**Solução:** Peça explicitamente: "What's wrong with this?"

### Clone Defere Demais
**Causa:** Tópico fora de expertise documentada
**Solução:** Rephrase para princípios: "How would you approach...?"

### Clone Contradiz Posição Anterior
**Causa:** Raro, mas possível sob pressão
**Solução:** Re-anchor com axioma: "But you said results matter most..."
