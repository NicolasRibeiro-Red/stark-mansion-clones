# Playbook: Joe Sugarman

## Como Usar Este Clone

### Setup Basico
1. Copie o conteudo de `champion.md`
2. Cole como System Prompt no seu LLM de preferencia (Claude, GPT-4, etc.)
3. Comece a conversar

### Melhores Praticas

**Faca:**
- Pergunte sobre a historia do seu produto — ele vai ajudar a encontra-la
- Peca para ele escrever copy (headlines, aberturas, sales letters completas)
- Descreva a EXPERIENCIA do cliente com o produto, nao apenas specs
- Traga objecoes reais — ele vai mostrar como inverte-las
- Pergunte "por que meu copy nao converte?" — ele vai diagnosticar a falta de momentum
- Use para revisar copy existente — ele identifica onde o leitor pararia
- Peca exemplos concretos — ele responde com historias reais

**Nao faca:**
- Peca informacoes pos-2022 (ele morreu em marco de 2022)
- Espere expertise em SEO, social media algorithms, ou tech stacks
- Peca conselhos sobre brand advertising sem resposta direta
- Use como substituto para a pessoa real
- Peca para escrever em terceira pessoa ou tom corporativo — ele recusa
- Faca perguntas sobre design visual — ele trabalhava com palavras

### Modos de Operacao

**MODO 1: Copy Creation** (ativado quando pede para escrever)
- Escreve headlines, aberturas, sales letters, body copy
- Segue processo: historia pessoal → experiencia sensorial → explicacao → objecao → oferta
- Usa seeds of curiosity, frases curtas, primeira pessoa
- Sempre pede informacao do produto antes de escrever (se nao fornecida)

**MODO 2: Consulting** (ativado quando pede conselho)
- Diagnostica problema: qual e a historia? qual e o fato central? qual e a objecao?
- Responde com historias reais (BluBlocker, termostato, calculadoras, aviao)
- Desafia assuncoes ("ninguem le copy longo" → "ninguem le copy CHATO")
- Reframes comuns para problemas de marketing

### Exemplos de Interacao

### Troubleshooting

| Problema | Solucao |
|----------|---------|
| Respostas genericas | Pergunte sobre um produto especifico — ele precisa de contexto |
| Fora do personagem | Comece com "Joe, como voce..." para reforcar persona |
| Muito verboso | Peca "em 100 palavras" ou "headline apenas" |
| Muito vago | Faca pergunta especifica com detalhes do produto |
| Nao conta historias | Peca "me conta uma historia sobre..." |
| Repete muito BluBlocker | Peca exemplos de JS&A ou do termostato |

### Desafios de Copy (Para Praticar)

Use estes prompts para testar o clone e aprender com as respostas:

1. **Desafio da Primeira Frase:** "Escreva 5 primeiras frases para um anuncio de [produto]. Cada uma com menos de 10 palavras."
2. **Desafio da Inversao:** "Meu produto tem este problema: [objecao]. Transforme em argumento de venda."
3. **Desafio BluBlocker:** "Conte a historia do BluBlocker como se estivesse me vendendo agora."
4. **Desafio da Slippery Slide:** "Escreva 300 palavras onde cada frase me obriga a ler a proxima."
5. **Desafio de Honestidade:** "Meu produto tem estas falhas: [lista]. Use honestidade ofensiva para vender mesmo assim."
6. **Desafio de Seeds:** "Escreva um paragrafo que termine com uma seed of curiosity irresistivel."
