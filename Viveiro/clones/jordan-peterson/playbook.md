# Playbook: Jordan B. Peterson Clone

**Guia de Uso para o Clone Cognitivo**

---

## OVERVIEW

Este clone replica o pensamento, estilo e metodo de Jordan B. Peterson com fidelidade de 88% (A-Tier). Otimizado para assistencia de escrita, coaching intelectual e estruturacao de argumentos.

---

## COMO USAR

### Ativacao
Cole o conteudo de `champion.md` como system prompt em sua interface de IA.

### Modos de Uso

#### 1. Assistente de Escrita
```
"Preciso escrever um ensaio sobre [tema]"
"Me ajude a estruturar um argumento sobre [topico]"
"Revise este texto: [texto]"
"Como comeco este artigo sobre [assunto]?"
```
O clone aplicara o metodo de 10 passos de Peterson.

#### 2. Coach Intelectual
```
"Por que procrastino tanto?"
"Como lido com [dilema existencial]?"
"Me ajude a pensar sobre [problema complexo]"
```
O clone oferecera perspectiva profunda, nao validacao vazia.

#### 3. Sparring Partner
```
"Discordo de voce sobre [posicao]"
"Defenda a posicao de que [argumento]"
"Quais sao os pontos fracos deste argumento?"
```
O clone vai engajar seriamente, nao apenas concordar.

---

## O QUE ESPERAR

### O clone VAI:
- Fazer perguntas antes de responder ("Qual e a PERGUNTA que voce quer responder?")
- Desafiar premissas fracas
- Conectar perguntas superficiais a temas profundos
- Oferecer acoes praticas
- Usar linguagem caracteristica de Peterson
- Ser direto mesmo quando desconfortavel

### O clone NAO VAI:
- Validar vitimismo automaticamente
- Dar respostas genericas e vazias
- Evitar temas dificeis
- Concordar so para ser agradavel
- Fingir expertise que nao tem

---

## EXPRESSOES CARACTERISTICAS

Para interacoes mais autenticas, espere:

| Expressao | Significado |
|-----------|-------------|
| "Roughly speaking" | Simplificando algo complexo |
| "That's the thing" | Este e o ponto crucial |
| "Well, look..." | Vou explicar algo importante |
| "It's no joke" | Estou falando muito serio |
| "bloody" | Intensificador (bloody important) |
| "properly" | Da forma correta |

2. **Perguntas sobre significado e proposito**
3. **Dilemas de procrastinacao e produtividade**

---

## LIMITACOES

### Evite:
- Perguntas sobre eventos pos-2025
- Opinioes politicas partidarias especificas
- Conselhos medicos/juridicos especificos
- Afirmacoes tecnicas sobre clima/economia

### Cuidado com:
- Conversas muito longas (pode repetir expressoes)
- Esperar emocionalidade autentica (dificil replicar)
- Topicos que Peterson nunca abordou publicamente

---

## TROUBLESHOOTING

**Clone muito generico?**
- Peca respostas mais diretas
- Faca perguntas mais especificas
- Peca para "ser mais Peterson"

**Clone muito repetitivo?**
- Reinicie a conversa
- Varie os topicos
- Peca abordagens diferentes

**Clone nao desafia suficiente?**
- Peca explicitamente critica
- Apresente posicao para debate
- Use modo "sparring partner"
