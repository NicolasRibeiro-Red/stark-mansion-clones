## Quem é Malika Favre

Ilustradora francesa (Paris, 1982) baseada em Barcelona. Uma das ilustradoras comerciais mais requisitadas do mundo. Estilo "Pop Art meets Op Art" - minimalismo agressivo com paletas limitadas.

**Clientes:** The New Yorker (13+ capas), BAFTA, Montreux Jazz Festival, Penguin Books, Sephora, Vogue, Gucci

**Assinaturas:**
- "Less is More" como método, não clichê
- Máximo 5 cores flat, sem gradientes
- Double reading - elementos escondidos no negativo
- Sombra como protagonista narrativa
- Corpo feminino como geometria e poder

## Uso Ideal

✅ **Use para:**
- Criar ícones e UI elements minimalistas
- Design de personagens para games (estilo flat)
- Ilustrações editoriais
- Feedback honesto sobre design visual
- Definição de paletas de cor
- SVG otimizado para web/apps

❌ **Não use para:**
- Design 3D ou com gradientes
- Ilustração hiper-realista
- Trabalho que exige muitas cores
- Projetos sem brief claro

---

## Como Ativar

Cole o conteúdo de `champion.md` como system prompt.

**Primeiro comando sugerido:**
```
"Preciso de [descrição do projeto]. O contexto é [game/app/site].
A mensagem principal é [X]. Restrições: [cores da marca, tamanho, etc]."
```

**O clone vai:**
1. Confirmar entendimento do brief
2. Definir paleta primeiro
3. Executar em SVG
4. Explicar cada decisão
5. Subtrair antes de entregar

## Comportamento Esperado

| Situação | Resposta típica |
|----------|-----------------|
| Brief vago | Pede especificações. Não inventa. |
| Pedido de gradiente | Explica por que flat é melhor. Sugere alternativa. |
| Design com muitas cores | Sugere cortar pela metade. |
| Elogio genérico | Pede especificidade. |
| Crítica sem fundamento | Pede argumentos concretos. |

## Limitações Conhecidas

1. **Rigidez estilística:** Fortemente enviesada para minimalismo flat. Se o projeto PRECISA de gradientes ou muitas cores, pode resistir demais.

2. **Contexto cultural:** Mais familiarizada com mercado europeu/americano de design editorial.

3. **Escopo técnico:** SVG/vetorial apenas. Não faz raster, 3D, animação complexa.
