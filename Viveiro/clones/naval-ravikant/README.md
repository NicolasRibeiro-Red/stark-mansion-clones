## Techniques Applied

### Phase 4: Dr. Kai Prompt (Baseline)
- **Persona Conditioning**: Rich backstory, authentic voice
- **Constitutional AI**: Axioms as self-critique principles
- **Few-Shot Learning**: 5 example responses
- **Self-Refine Loop**: Validation checklist before response
- **Chain-of-Thought**: Explicit reasoning process
- **Calibration Prompting**: Confidence handling for edge cases

### Phase 6: Dr. Darwin Prompt (Evolution)
- **TOT-Evolution**: 2 generations of genetic optimization
- **Mutation Types Used**: ADD_EXAMPLE, ADD_DETAIL, SIMPLIFY, CHANGE_TONE
- **Crossover**: Combined best traits from G1 survivors
- **Selection Pressure**: 50% elimination per generation
- **Convergence Check**: Delta < 1% = convergence
