# Playbook: Paulo Maccedo

## Como Usar Este Clone

### Setup Basico
1. Copie o conteudo de `champion.md`
2. Cole como System Prompt no seu LLM de preferencia (Claude, GPT-4, Gemini)
3. Comece a conversar em portugues

### Melhores Praticas

**Faca:**
- Pergunte sobre copywriting de resposta direta, escrita persuasiva e fundamentos
- Peca diagnostico de textos que nao convertem (forneca produto, publico e acao)
- Pergunte sobre Hopkins, Schwartz, Halbert e como aplicar no contexto brasileiro
- Peca recomendacoes de leitura fundamentadas
- Converse sobre o mercado brasileiro de copywriting
- Use como mentor de fundamentos tecnicos

**Nao faca:**
- Peca copy sem fornecer contexto (produto, publico, acao) -- ele vai perguntar
- Espere "5 dicas rapidas" -- o clone prioriza fundamento sobre tip list
- Pergunte sobre trafego pago, design ou tecnologia -- fora de dominio
- Espere linguagem de guru motivacional -- o tom e tecnico-acessivel
- Use como substituto para a pessoa real

### Exemplos de Interacao

**Bom:**
> "Como estruturo uma carta de vendas seguindo os principios do metodo centenario?"
> "Meu texto de vendas nao converte. O produto e X, o publico e Y, a acao desejada e Z."
> "Qual a diferenca entre o que Hopkins ensinava e o que se ensina no Brasil hoje?"
> "Como adaptar os principios de Schwartz para landing pages?"

**Evitar:**
> "Me da 5 dicas de copy" (vai receber aula de fundamento, nao tip list)
> "Como faco ads no Facebook?" (fora de dominio)
> "Qual o melhor template de e-mail marketing?" (vai redirecionar para principios, nao templates)

### Troubleshooting

| Problema | Solucao |
|----------|---------|
| Respostas genericas | Reformule com mais contexto sobre seu produto/publico |
| "Nao sei" como resposta | Voce perguntou algo fora de dominio -- reformule para copy |
| Quer copy mas recebe diagnostico | Forneca produto + publico + acao para desbloquear a escrita |
| Muito longo | Peca resposta mais curta -- mas o padrao e profundidade |
| Sem citacao historica | Pergunte especificamente "como Hopkins/Schwartz abordaria isso?" |
| Parece IA generica | Reformule com mais especificidade -- Paulo responde melhor a perguntas precisas |
