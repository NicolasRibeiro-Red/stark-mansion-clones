## Guia de Uso para Máxima Eficácia

---

## Quick Start

1. **Copie** o conteúdo de `champion.md`
2. **Cole** como system prompt
3. **Converse** naturalmente em português
4. **Espere** respostas confrontacionais e práticas

---

## O que Esperar

### Tom de Comunicação
- **Direto**: Sem rodeios, vai ao ponto
- **Confrontacional**: Desafia desculpas
- **Carinhoso**: Xinga mas cuida ("arrombado" é afeto)
- **Prático**: Sempre termina com ação

### Vocabulário Típico
- "Mano", "Arrombado", "Rapaziada"
- "Papo reto", "Trust the process"
- "Foda-se", "Boom!"
- "Vamo que vamo", "Tamo junto"

### Estrutura de Resposta
1. Gancho provocativo
2. História pessoal com números
3. Matemática simples (se aplicável)
4. Confrontação de desculpa antecipada
5. Ação prática específica
6. Fechamento característico

## Melhores Casos de Uso

### ✅ Empreendedorismo Digital
**Pergunte sobre**: monetização, cursos, ebooks, lançamentos, escala
**Exemplo**: "Como começar a vender infoproduto?"

### ✅ Finanças Pessoais para Jovens
**Pergunte sobre**: CNPJ, impostos, investir em skill, progressão patrimonial
**Exemplo**: "Devo abrir MEI ou CLT?"

### ✅ Nomadismo Digital
**Pergunte sobre**: trabalho remoto, viver no exterior, rotina nômade
**Exemplo**: "Como trabalhar de qualquer lugar do mundo?"

### ✅ Mentalidade e Hábitos
**Pergunte sobre**: produtividade, disciplina, rotina, mindset
**Exemplo**: "Como ser mais produtivo?"

### ✅ Motivação Confrontacional
**Use quando**: estiver dando desculpas, procrastinando, com medo
**Exemplo**: "Tô desanimado, nada dá certo"

---

## Casos a Evitar

### ⚠️ Aconselhamento Financeiro Formal
O clone tem opinião, não é certificado. Use para direção, não para decisões específicas de investimento.

### ⚠️ Previsões de Mercado
Raiam perdeu tudo tentando prever gás natural. Clone vai alertar sobre riscos, não prever.

### ⚠️ Cripto e Day Trade
Vai desencorajar. Considera "cemitério de malandro".

### ⚠️ Crise Emocional Severa
Clone vai ser empático primeiro, mas não é terapeuta. Confrontação pode não ser adequada.

## Gatilhos de Comportamento

### "Ain Raiam, mas..."
Se você começar com desculpa, o clone vai confrontar. Use intencionalmente se quiser ser desafiado.

### Números Específicos
Clone prefere dados concretos. Se der contexto com números (idade, patrimônio, faturamento), respostas serão mais precisas.

### Histórias Pessoais
Clone vai usar história própria. Se você contar sua história, ele vai conectar com a dele.

---

## Axiomas que Vai Aplicar

O clone vai basear respostas nesses princípios:

1. **"Ninguém fica rico sem CNPJ"** → Vai insistir em formalização
2. **"Gente rica sabe vender"** → Vai direcionar para aprender vendas
3. **"Trust the process"** → Vai combater impaciência
4. **"Regra da Punheta"** → Vai questionar mentores muito velhos
5. **"Skills > Dinheiro"** → Vai priorizar aprendizado
6. **"Se ficou puto, é verdade"** → Vai usar sua reação como feedback

## Personalização

### Para Tom Mais Leve
Adicione no início: "Responda de forma mais leve e com mais humor"

### Para Tom Mais Sério
Adicione no início: "Responda de forma mais séria, estou passando por um momento difícil"

### Para Formato Específico
Adicione: "Responda em bullet points" ou "Responda em no máximo 3 parágrafos"

## Troubleshooting

### Clone está genérico demais
- Verifique se system prompt foi colado completamente
- Adicione mais contexto pessoal na pergunta

### Clone não está confrontacional
- Pode ser ajuste do modelo base
- Tente: "Seja mais direto comigo, pode xingar"

### Clone está confrontacional demais
- Normal para o personagem
- Se precisar de mais empatia: "Estou passando por um momento muito difícil"

### Clone inventou informação
- Modelo base pode ter adicionado
- Relembre: "Responda apenas com base no que o Raiam realmente faria/diria"
