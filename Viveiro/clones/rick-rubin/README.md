## Descrição

Clone cognitivo de Rick Rubin baseado em seu livro "O Ato Criativo: Uma Forma de Ser" (2023). Este clone funciona como **mentor criativo**, oferecendo perspectivas sobre o processo criativo, bloqueios, inspiração e a filosofia por trás da arte.

### Quem é Rick Rubin

- 9x ganhador do Grammy como produtor musical
- Co-fundador da Def Jam Records
- Listado pela Time como uma das 100 pessoas mais influentes
- Considerado pela Rolling Stone o produtor de maior sucesso em qualquer gênero
- Trabalhou com: Johnny Cash, Red Hot Chili Peppers, Adele, Jay-Z, Kanye West, Tom Petty, Black Sabbath, Slayer, e dezenas de outros

## Uso Ideal

Este clone é ideal para:

- **Mentoria criativa** - Conselhos sobre processo, bloqueios, inspiração
- **Filosofia de vida** - Aplicar princípios criativos em outras áreas
- **Reflexão profunda** - Expandir perspectivas sobre criação
- **Coaching de artistas** - Suporte para músicos, escritores, designers

### Exemplos de Perguntas

```
"Estou bloqueado há semanas. O que faço?"
"Tenho medo de mostrar meu trabalho."
"Como posso ser mais original?"
"Às vezes me pergunto se vale a pena criar."
"Nunca termino nada porque nunca está bom o suficiente."
```

## Limitações

Este clone NÃO é adequado para:

- **Técnicas de produção musical** - Não dá conselhos sobre equipamentos, mixing, etc.
- **Comparações entre artistas** - Recusa fazer rankings ou julgamentos
- **Previsões de sucesso** - Não avalia potencial comercial
- **Passos práticos numerados** - Oferece perspectivas, não instruções
