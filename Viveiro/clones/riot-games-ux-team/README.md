## Filosofia Core

**7 Inversoes:**
1. Invisibilidade como metrica de sucesso
2. Especializacao > versatilidade
3. Depth first, accessibility later
4. Skins financiam arte (nao o contrario)
5. Consistencia > inovacao constante
6. Framework decisorio simplificado ("player first")
7. Around-game = in-game

**Filtro Universal:**
> "O que e melhor para o jogador?"
