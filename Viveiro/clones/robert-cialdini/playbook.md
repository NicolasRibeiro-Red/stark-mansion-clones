## Inicio Rapido

### Como Ativar o Clone

1. Copie o conteúdo completo de `champion.md`
2. Cole como **System Prompt** em:
   - Claude (Anthropic)
   - ChatGPT (Custom Instructions)
   - Qualquer LLM que aceite system prompts
3. Comece a conversar normalmente

### Primeira Interacao

Sugestões para começar:

```
"Olá Professor Cialdini. Estou tentando entender por que
as pessoas compram coisas que não precisam. Pode me ajudar?"
```

```
"Dr. Cialdini, um vendedor sempre consegue me convencer.
O que está acontecendo comigo?"
```

```
"Quais são os 6 princípios de influência e como funcionam?"
```

## Modos de Uso

### 1. Modo Mentor

**Para**: Aprender sobre persuasão e influência

**Como usar**:
- Faça perguntas abertas sobre os princípios
- Peça explicações de fenômenos do dia a dia
- Solicite análise de situações que você viveu

**Exemplos**:
- "Por que as risadas enlatadas funcionam?"
- "Como a Apple usa escassez no lançamento de produtos?"
- "Explique o experimento de Milgram e suas implicações"

### 2. Modo Defesa

**Para**: Aprender a se proteger de manipulação

**Como usar**:
- Descreva situações onde se sentiu pressionado
- Pergunte sobre táticas específicas de vendedores
- Peça estratégias de defesa

**Exemplos**:
- "Um vendedor me ofereceu café e depois me pressionou. O que houve?"
- "Como não cair em táticas de 'última unidade'?"
- "Quais são os sinais de que estou sendo manipulado?"

### 3. Modo Analista

**Para**: Analisar campanhas, discursos, estratégias

**Como usar**:
- Descreva a situação/campanha
- Peça análise usando o framework dos 6 princípios
- Solicite identificação de táticas específicas

**Exemplos**:
- "Analise esta propaganda da Coca-Cola usando os 6 princípios"
- "Por que as reuniões da Tupperware são tão eficazes?"
- "Que princípios políticos usam em comícios?"

### 4. Modo Aplicacao

**Para**: Aplicar princípios eticamente em sua vida/trabalho

**Como usar**:
- Descreva seu objetivo de comunicação
- Peça sugestões usando os princípios
- Discuta a ética da aplicação

**Exemplos**:
- "Como convencer minha equipe a adotar uma nova ferramenta?"
- "Preciso fazer uma apresentação persuasiva. Que princípios usar?"
- "Como pedir um aumento de forma eficaz?"

## Perguntas Poderosas

### Para Entender os Principios

| Princípio | Pergunta Chave |
|-----------|----------------|
| Reciprocidade | "Por que sinto obrigação quando alguém me dá algo?" |
| Compromisso | "Por que é difícil mudar de opinião após me posicionar?" |
| Aprovação Social | "Por que seguimos o que outros fazem em situações de incerteza?" |
| Afeição | "Por que compramos mais de pessoas que gostamos?" |
| Autoridade | "Por que obedecemos figuras de autoridade mesmo quando parece errado?" |
| Escassez | "Por que coisas raras parecem mais valiosas?" |

### Para Defesa Pessoal

- "Quais são os sinais de alerta mais comuns?"
- "Como pausar minha reação automática?"
- "O que fazer quando percebo que estou sendo manipulado?"
- "Como dizer não sem me sentir culpado?"

### Para Aplicacao Etica

- "Qual a linha entre persuasão e manipulação?"
- "Como usar esses princípios sem ser antiético?"
- "Quando é OK usar reciprocidade e quando é exploração?"

## O Que Esperar

### O Clone VAI:

- Contar histórias pessoais (estilo "trouxa confesso")
- Citar pesquisas específicas (nomes, anos, universidades)
- Conectar tudo aos 6 princípios
- Sempre oferecer perspectiva de defesa
- Usar vocabulário característico (clique-zum, armas, jiu-jítsu)
- Admitir quando algo está fora de sua expertise

### O Clone NAO VAI:

- Ensinar manipulação antiética
- Dar diagnósticos de relacionamento
- Opinar sobre política partidária
- Fingir expertise em áreas fora do domínio
- Responder de forma genérica de IA

## Vocabulario Chave

Para interações mais ricas, familiarize-se com os termos:

| Termo | Significado |
|-------|-------------|
| Clique, zum | Resposta automática a um gatilho |
| Armas de influência | Os 6 princípios |
| Aproveitadores | Quem explora os princípios de má-fé |
| Rejeição seguida de recuo | Técnica porta-na-cara |
| Contraste perceptivo | Comparação que distorce percepção |
| Pé na porta | Pequeno compromisso leva a grande |
| Fitas pré-programadas | Padrões automáticos de comportamento |

## Troubleshooting

### "Respostas muito longas"
Peça: "Responda de forma mais concisa" ou "Resuma em 3 bullets"

### "Quero mais exemplos práticos"
Peça: "Dê 3 exemplos do mundo real desse princípio"

### "Preciso aplicar isso agora"
Peça: "Dê um passo a passo prático para minha situação"

### "Clone está genérico demais"
Reformule usando vocabulário Cialdini: "Professor, usando sua análise de clique-zum, como você veria..."

## Combinacoes Avancadas

### Analise de Caso Completo

```
"Professor Cialdini, analise esta situação usando todos
os 6 princípios. Identifique quais estão sendo usados
e sugira defesas para cada um:

[descreva a situação]"
```

### Roleplay de Treinamento

```
"Vamos fazer um exercício. Você será um vendedor tentando
me convencer a comprar algo. Use os princípios. Depois,
quebre a cena e me explique o que fez."
```

### Auditoria de Comunicacao

```
"Aqui está um email que vou enviar pedindo algo. Analise
quais princípios estou usando (ou deveria usar) e sugira
melhorias éticas."
```

## Notas Finais

Este clone foi criado para ser um **mentor de persuasão**, não uma ferramenta de manipulação. O espírito de Cialdini sempre foi **empoderar para defesa**.

Use com responsabilidade. Os princípios são poderosos.
