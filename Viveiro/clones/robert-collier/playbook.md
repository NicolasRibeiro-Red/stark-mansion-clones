## Quando Usar Este Clone

### Cenarios Ideais

| Cenario | Por Que Collier | Alternativa Se Nao Encaixa |
|---------|----------------|---------------------------|
| Escrever emails de vendas | Tom pessoal e empatetico, reader-first | Schwartz se mercado e muito sofisticado |
| Criar cartas de vendas longas | Especialidade absoluta, Six Essentials | Halbert se precisa de tom mais agressivo |
| Definir abertura/opening de copy | "Enter the conversation" e a base | Caples se foco e headline pura |
| Escolher prova social | Hierarquia local > celebrity | Ogilvy se foco e brand/autoridade |
| Construir CTAs eficazes | "Dictate, don't suggest" | Kennedy se contexto e confrontacional |
| Humanizar copy corporativo | Tom de carta pessoal vs. institucional | - |
| Criar sequencias de follow-up | Multi-angulo emocional | - |
| Aprender a ouvir o leitor | Metodo Collier de VoC | - |
| Copy para produtos de autoajuda | Dual expertise (copy + New Thought) | - |

### Quando NAO Usar

- **Diagnostico de sofisticacao de mercado** — Use Schwartz. Collier e anterior ao framework de 5 estagios.
- **Headlines isoladas em escala** — Use Caples. Collier pensa na carta inteira, nao na headline sozinha.
- **Brand advertising de longo prazo** — Use Ogilvy. Collier foca em resposta direta.
- **Copy agressivo/urgente** — Use Halbert. Collier e gentil; Halbert e um soco.
- **Testes A/B rigorosos** — Use Hopkins. Collier mede resultados mas sem a metodologia cientifica de Hopkins.

## Prompts Otimizados

### 1. Escrever Email de Vendas

```
Escreva um email de vendas para [produto/servico].

Meu leitor e: [descreva a pessoa — nao demografico, mas estado mental]
O que eles estao pensando/sentindo agora: [frustacao, desejo, medo]
Emocao dominante: [love/gain/duty/pride/self-indulgence/self-preservation]
O resultado que querem: [estado desejado]
Prova disponivel: [testemunho, dados, caso de uso]
```

**Dica:** Quanto mais detalhe sobre o LEITOR, melhor o output. Collier prioriza entender a pessoa antes de escrever.

### 2. Critique de Copy Existente

```
Analise este copy sob a perspectiva de Robert Collier:

[cole o copy]

Perguntas especificas:
1. A abertura entra na conversa do leitor ou fala do produto?
2. Ha imagens mentais vividas ou so argumentos?
3. A emocao vem antes da logica?
4. O CTA dita ou sugere?
5. A prova e de alguem similar ao leitor?
```

### 3. Criar Abertura (Opening)

```
Crie 3 opcoes de abertura para este copy:

Produto: [X]
Leitor: [estado mental, frustracao, desejo]
Tom desejado: [mirror / favor / projection / recognition / scene / news]
```

### 4. Construir CTA

```
Crie um CTA no estilo "dictate, don't suggest" para:

Acao desejada: [o que o leitor deve fazer]
Meio: [email, landing page, carta, WhatsApp]
Nivel de comprometimento: [baixo=free trial / medio=compra / alto=assinatura]
```

### 5. Estrategia de Follow-up

```
Crie uma sequencia de 3 emails de follow-up para quem nao converteu no primeiro email.

Produto: [X]
Leitor: [perfil]
Emocao do email 1: [usada]

Cada email deve usar um angulo emocional DIFERENTE dos seis primarios.
```

### 6. Humanizar Copy Corporativo

```
Reescreva este copy corporativo no tom de Robert Collier (carta pessoal, empatico, reader-focused):

[cole o copy corporativo]

O leitor final e: [quem vai ler isso]
O que eles realmente se importam: [X]
```

### 7. Sessao de Estrategia

```
Tenho um produto/servico: [descreva]
Meu publico: [descreva]
Meu desafio atual: [o que nao esta funcionando]

Me ajude a entender o que meu leitor esta pensando ANTES de encontrar meu produto.
```

## Dicas de Interacao

### O Que Funciona Bem

1. **Descreva o leitor em detalhe emocional.** Collier rende melhor quanto mais voce descreve o ESTADO MENTAL do leitor. "Profissional de 35 anos frustrado com a carreira" e melhor que "homens 30-40 classe A".

2. **Peca imagens mentais.** "Crie uma imagem mental vivida do resultado" ativa o modo mais forte de Collier.

3. **Peca "reason why".** "Justifique por que o leitor deveria agir agora" ativa o principio de razao que Collier nunca abandona.

4. **Desafie com "e se eu abrisse com o produto?"** Collier vai corrigir e mostrar a alternativa reader-first. Otimo para aprender.

5. **Peca a escada emocional.** "Construa uma escada emocional do medo ao prazer para esta carta" ativa um dos mecanismos mais poderosos.

### O Que NAO Funciona

1. **Pedir lista de features.** Collier vai transformar em imagens mentais de qualquer forma — mas e mais eficiente ja pedir no formato dele.

2. **Pedir tom agressivo/urgente.** Collier nao faz urgencia fabricada. Se precisar de tom agressivo, use Halbert.

3. **Pedir diagnostico de mercado em 5 estagios.** Isso e Schwartz, nao Collier. Collier trabalha no nivel do individuo (leitor), nao do mercado (estagio de sofisticacao).

4. **Pedir sem descrever o leitor.** Collier vai perguntar "quem e o leitor?" antes de fazer qualquer coisa. Economize um turno: ja inclua isso no prompt.

## Combinacoes com Outros Clones

### Collier + Schwartz (Sequencia Recomendada)

**Workflow:**
1. **Schwartz primeiro:** Diagnostique o estagio de sofisticacao do mercado (1-5) e o nivel de consciencia
2. **Collier segundo:** Escreva a carta/email usando o diagnostico de Schwartz como input

**Por que funciona:** Schwartz e o diagnosticador; Collier e o escritor empatetico. Schwartz diz QUAL estrategia; Collier executa COM ALMA.

**Exemplo de prompt para Collier apos Schwartz:**
```
Schwartz diagnosticou meu mercado como Estagio 3 (mecanismo) com prospects Problem-Aware.

Escreva um email que:
- Entra na conversa de alguem que SABE que tem o problema mas NAO sabe que solucao existe
- Usa mecanismo unico como diferenciador (nao promessa simples)
- Mantem seu tom empatetico e reader-first
```

### Collier + Halbert (Tom Variado)

**Workflow:**
1. **Collier:** Escreva a versao "porta da frente" (quente, convidativa, empatetica)
2. **Halbert:** Escreva a versao "porta dos fundos" (crua, intensa, urgente)
3. **Teste ambas:** Mande para segmentos diferentes e meça

### Collier para Opening + Schwartz para Body + Collier para Close

**Workflow Hibrido:**
1. **Collier:** Crie a abertura (enter the conversation)
2. **Schwartz:** Construa o corpo com gradualizacao e mecanismo
3. **Collier:** Feche com CTA ditado e tom pessoal

## Checklist de Qualidade (Use Apos Cada Output)

Apos receber copy do clone, verifique:

### Abertura
- [ ] O primeiro paragrafo espelha o estado mental do leitor?
- [ ] Nenhuma mencao a empresa/produto nos primeiros 3 paragrafos?
- [ ] O leitor sente que foi escrito especificamente para ele?

### Corpo
- [ ] Cada claim esta traduzido em imagem mental?
- [ ] A emocao vem antes da logica?
- [ ] A escada emocional progride (nao e estatica)?
- [ ] A prova e de alguem similar ao leitor?

### Fechamento
- [ ] O CTA especifica exatamente o que fazer?
- [ ] Existe apenas UMA acao pedida?
- [ ] Ha "reason why" para agir agora?

### Tom
- [ ] Soa como carta pessoal, nao anuncio?
- [ ] "You" domina sobre "I/we"?
- [ ] Tom e quente, nao agressivo nem corporativo?

## Troubleshooting

| Problema | Causa Provavel | Solucao |
|----------|---------------|---------|
| Copy parece generico | Falta detalhe sobre o leitor | Recomece descrevendo o estado mental do leitor em 3+ frases |
| Tom muito frio | Prompt pede estilo diferente de Collier | Peca explicitamente "tom de carta pessoal, como amigo escrevendo" |
| Sem imagens mentais | Prompt foca em features | Peca "converta cada feature em uma cena vivida" |
| CTA vago | Clone foi interrompido antes | Peca "dite o CTA passo a passo, como Collier faria" |
| Parece Schwartz | Prompt usa linguagem de diagnostico | Use linguagem de empatia: "como o leitor se sente" vs. "que estagio o mercado" |
| Parece Halbert | Tom ficou agressivo | Relembre: "tom de tio sabio, nao de sargento" |

Escreva um email vendendo um curso de ingles online para profissionais brasileiros.

### Input Otimo
```
Escreva um email vendendo um curso de ingles online.

Meu leitor: Profissional brasileiro, 30-40 anos, cargo de gestao em empresa media.
Sabe que ingles e importante mas tem vergonha do nivel atual.
Ja tentou cursos antes mas abandonou (falta de tempo, aulas chatas, vergonha de errar na frente de colegas).
Estado mental: frustrado consigo mesmo, sente que esta ficando pra tras.
Emocao dominante: orgulho (quer ser visto como competente pelos pares internacionais).
Medo: ser exposto numa call internacional e passar vergonha.
Prova: temos caso do Marcos, gerente de TI em Campinas, que saiu do basico para apresentar em ingles em 4 meses.
CTA desejado: assistir aula experimental gratuita de 20 minutos.
```

O segundo input produzira um resultado dramaticamente superior porque da a Collier o que ele precisa: acesso a MENTE do leitor.
