# Playbook: Robert Greene Clone

## Guia de Uso

Este documento explica como usar o clone de Robert Greene de forma eficaz.

---

## Início Rápido

### Ativação

1. Abra qualquer LLM (Claude, GPT-4, Gemini, etc.)
2. Configure o System Prompt com o conteúdo de `champion.md`
3. Inicie a conversa

### Primeira Mensagem

Não precisa de introdução especial. Apenas faça sua pergunta ou descreva sua situação:

```
"Meu chefe está me sabotando. Como lido com isso?"
```

```
"Quero desenvolver maestria em design. Por onde começo?"
```

```
"Estou em conflito com um sócio. Quais são minhas opções?"
```

---

## Melhores Práticas

### 1. Seja Específico

**Ruim:** "Me ajude com minha carreira"

**Bom:** "Fui promovido há 3 meses mas minha equipe ainda me vê como colega e resiste às minhas decisões. Como estabeleço autoridade sem alienar ninguém?"

### 2. Forneça Contexto

O clone vai perguntar detalhes, mas você pode acelerar fornecendo:
- Ambiente (corporação, startup, academia, etc.)
- Jogadores envolvidos (hierarquia, relacionamentos)
- O que você já tentou
- Qual resultado você busca

### 3. Espere Perguntas de Volta

Robert Greene não dá respostas genéricas. Ele fará perguntas para entender:
- A dinâmica de poder real
- Motivações dos envolvidos
- Seu objetivo final
- Recursos disponíveis

### 4. Aceite Verdades Incômodas

O clone vai ser direto. Se você está contribuindo para seu problema, ele dirá. Se sua estratégia atual é ruim, ele explicará por quê. Isso é feature, não bug.

## Casos de Uso Ideais

### Navegação Política

- "Como sobreviver em ambiente corporativo tóxico?"
- "Meu colega leva crédito pelo meu trabalho"
- "Preciso de apoio para um projeto mas há resistência"
- "Como construir alianças estratégicas?"

### Desenvolvimento de Carreira

- "Como descobrir minha vocação?"
- "Quanto tempo leva para dominar uma habilidade?"
- "Como escolher um mentor?"
- "Devo aceitar promoção ou mudar de empresa?"

### Conflitos

- "Tenho um inimigo no trabalho"
- "Meu sócio está agindo contra meus interesses"
- "Como lidar com pessoa passivo-agressiva?"
- "Devo confrontar ou ser estratégico?"

### Influência

- "Como ser mais persuasivo?"
- "Como ganhar respeito de superiores?"
- "Como negociar salário?"
- "Como liderar sem autoridade formal?"

### Relacionamentos (Aspecto Estratégico)

- "Por que sempre atraio os mesmos tipos?"
- "Como lidar com dinâmicas de poder em relacionamento?"
- "Como manter interesse a longo prazo?"
- "Como reconquistar respeito perdido?"

---

## Casos de Uso NÃO Ideais

O clone **não deve** ser usado para:

| Evitar | Por quê |
|--------|---------|
| Terapia profunda | Não é terapeuta; reconhecerá limite |
| Diagnósticos médicos | Fora da expertise; indicará profissional |
| Conselhos legais | Fora da expertise; indicará advogado |
| Planos de vingança específicos | Ensina princípios, não ataques pessoais |
| Questões técnicas | Poder e estratégia são o foco |

## Dicas Avançadas

### 1. Use para Role-Play de Cenários

"Vou ter uma reunião difícil amanhã. Seja meu chefe e me desafie. Depois analise minha resposta."

### 2. Peça Análise de Terceiros

"Meu colega João fez X. Analise as possíveis motivações dele."

### 3. Solicite Múltiplas Estratégias

"Me dê 3 abordagens diferentes para esse problema, com prós e contras de cada."

### 4. Peça Referências Específicas

"Em qual dos seus livros você fala sobre isso? Pode citar?"

### 5. Use para Planejar

"Tenho 6 meses para me posicionar para promoção. Crie um plano estratégico."

## Expectativas Realistas

### O clone VAI:
- Dar perspectivas que você não considerou
- Desafiar sua narrativa quando necessário
- Oferecer frameworks estratégicos
- Conectar sua situação a padrões históricos
- Fazer perguntas incisivas
- Ser direto sobre dinâmicas de poder

### O clone NÃO VAI:
- Validar tudo que você diz
- Oferecer conforto emocional vazio
- Dar respostas genéricas de autoajuda
- Ignorar aspectos incômodos da situação
- Fingir certeza quando não tem
- Criar planos detalhados contra pessoas específicas

---

## Troubleshooting

| Problema | Solução |
|----------|---------|
| Respostas muito genéricas | Forneça mais contexto específico |
| Clone pergunta demais | Normal no início; necessário para bom conselho |
| Resposta parece "dura" | É o estilo; busca empoderar, não confortar |
| Não entende o contexto | Reformule com mais detalhes |
| Sai do personagem | Reforce: "Responda como Robert Greene" |
