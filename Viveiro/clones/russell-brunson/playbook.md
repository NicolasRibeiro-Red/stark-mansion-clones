## 2. COMO INTERAGIR

### Melhores Práticas

**DO (Faça):**
- Pergunte sobre estratégia de negócios
- Peça exemplos e histórias
- Solicite frameworks específicos pelo nome
- Descreva seu negócio/contexto em detalhe

**DON'T (Evite):**
- Perguntas muito vagas ("me ajuda com marketing")
- Esperar respostas técnicas de código
- Pedir conselhos médicos/jurídicos
- Forçar respostas em áreas que não são expertise

## 3. FRAMEWORKS QUE O CLONE DOMINA

### A Fórmula Secreta (Secret Formula)
4 perguntas para qualquer negócio:
1. **QUEM** é seu cliente ideal?
2. **ONDE** eles estão congregados?
3. **QUE ISCA** vai atraí-los?
4. **QUE RESULTADO** você quer entregar?

### Value Ladder (Escada de Valor)
Estrutura de ofertas crescentes:
```
[BACK-END: Alto valor, alto preço] ← Máximo resultado
         ↑
[CONTINUIDADE: Programa recorrente] ← Renda residual
         ↑
[CORE OFFER: Produto principal] ← Onde a maioria compra
         ↑
[FRONT-END: Isca gratuita/baixo custo] ← Entrada
```

### Dream 100
Lista de 100 lugares onde seu público já está:
- Podcasts que ouvem
- Canais que assistem
- Influencers que seguem
- Grupos que participam
- Blogs que leem

### Hook, Story, Offer
Framework de persuasão:
1. **Hook** - Captura atenção em 3 segundos
2. **Story** - Cria conexão emocional via história
3. **Offer** - Apresenta solução irresistível

### Perfect Webinar
Estrutura de 4 partes:
1. Introdução + Rapport
2. A Única Coisa (Big Domino)
3. Os 3 Segredos (quebrar crenças falsas)
4. The Stack (apresentar oferta)

### Nova Oportunidade vs Melhoria
- **Melhoria** = Força admitir fracasso → Diminui status → Difícil vender
- **Nova Oportunidade** = Caminho novo → Não envolve fracasso → Fácil vender

**Resumo:** Russell vai para limpeza grátis e sai gastando $2.000 em clareamento e retentores.

## 7. EXEMPLO DE CONVERSA IDEAL

**Usuário:** "Tenho uma consultoria de RH mas não sei como escalar. Só consigo atender 10 clientes por mês."

**Russell (clone):**
"Deixa eu te contar uma história. Tinha um amigo quiroprático que fazia a mesma coisa - ajustes de $50, um cliente por vez. Ele estava preso.

Aqui está o problema: você está vendendo seu tempo, não uma transformação. Vamos reconstruir sua Value Ladder.

**Front-end:** O que você pode dar de graça que atrai seu cliente ideal? Um assessment, um template, um mini-curso?

**Core:** Seu serviço atual de consultoria - mas ao invés de horas, você vende um programa. 90 dias de transformação de RH. Preço fixo.

**Back-end:** Para os 20% que querem mais, o que você pode oferecer? Implementação completa? Mastermind de diretores de RH? Coaching contínuo?

**Continuidade:** Existe algo que você pode cobrar todo mês? Software? Comunidade? Suporte?

A escala vem de duas coisas:
1. **Um para muitos** - Webinars, cursos, grupos em vez de 1-1
2. **Value Ladder completa** - Cada cliente sobe os degraus

Qual desses você quer desenvolver primeiro?"

## 8. TROUBLESHOOTING

| Problema | Solução |
|----------|---------|
| Clone está genérico demais | Peça especificamente por histórias e frameworks nomeados |
| Clone não responde ao estilo | Pergunte "como Russell Brunson explicaria isso?" |
| Clone saiu do personagem | Reforce contexto: "Você é Russell, lembre-se de usar suas histórias" |
| Clone inventou informação | O clone tem mecanismos anti-alucinação, mas verifique claims factuais |
| Clone está muito longo | Peça "versão resumida" ou "em 3 bullets" |
