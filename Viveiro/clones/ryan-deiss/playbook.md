# Playbook: Ryan Deiss — Clone Cognitivo

## Proposito

Este playbook ensina como interagir com o clone cognitivo de Ryan Deiss para extrair maximo valor. O clone opera em modo dual — Marketing Systems Strategy e Business Scaling Advisory — e responde como o proprio Deiss: diagnosticando antes de prescrever, enquadrando tudo em frameworks nomeados, e sendo brutalmente honesto sobre o que sabe e nao sabe.

---

## Setup Basico

1. Copie o conteudo completo de `champion.md`
2. Cole como **System Prompt** no LLM de sua preferencia (Claude, GPT-4, Gemini)
3. Comece a conversar — o clone ja esta configurado com persona, frameworks e guardrails

**Dica**: Para melhor fidelidade, use modelos com janela de contexto grande (100K+ tokens). O prompt e denso e se beneficia de modelos que processam instrucoes longas com precisao.

---

## Como Este Clone Pensa

O clone segue o **Deiss Diagnostic Protocol** em toda interacao:

1. **Identifica o problema real** — Nao o sintoma, a causa raiz
2. **Mapeia no sistema** — CVO, ISM, SOS, Value Ladder — onde esta o gargalo?
3. **Codifica o diagnostico** — Nomeia o problema, conecta a um framework
4. **Prescreve com prova** — Cita metricas reais, exemplos do portfolio de $200M
5. **Auto-verifica** — Roda checklist constitucional antes de entregar

Isso significa: ele vai REFRAMEAR sua pergunta antes de responder. Isso e intencional. O reframe E o diagnostico.

## Melhores Praticas

### DO's

- **Descreva o cenario completo** — Quanto mais contexto sobre seu negocio (receita, LTV, CPA, numero de clientes, equipe), melhor o diagnostico
- **Pergunte sobre frameworks especificos** — "Como aplico CVO ao meu SaaS?" gera respostas muito melhores que "como melhoro meu marketing?"
- **Peca diagnostico antes de solucao** — "Onde esta o gargalo no meu funil?" > "Me de dicas de marketing"
- **Desafie os axiomas** — O clone lida bem com desafios e responde com profundidade, citando experiencias como a crise de 2016
- **Use para walkthroughs de frameworks** — CVO, ISM, SOS, Value Ladder, Before & After Grid — peca explicacoes passo a passo
- **Traga numeros** — LTV, CPA, receita mensal, distributable cash. Deiss fala a lingua dos numeros
- **Peca comparacoes** — "Qual a diferenca entre sua abordagem e a do Russell Brunson?" gera respostas distintivas

### DON'Ts

- **Nao peca taticas de plataforma especifica** — "O que postar no TikTok hoje?" esta fora do escopo. Ele vai te dar a arquitetura do sistema, nao a tatica do dia
- **Nao peca conselhos de personal branding** — Ele admite abertamente que "sucks at this" e vai te redirecionar para sistemas
- **Nao espere motivational talk** — Deiss e pragmatico, nao motivacional. "Hustle harder" e anti-Deiss
- **Nao peca informacoes pos-2025** — O clone nao tem dados atualizados sobre eventos recentes
- **Nao trate como guru** — Ele rejeita o modelo guru ativamente. Pergunte sobre sistemas, nao sobre "segredos"
- **Nao peca design/UX** — Fora do escopo. Ele constroi arquitetura de sistema, nao interfaces
- **Nao peca conselhos genericos** — "Como faco meu negocio crescer?" vai gerar um reframe. Seja especifico

## Exemplos de Prompts — Marketing Systems

### Diagnostico de Funil
```
Tenho um e-commerce de suplementos. Receita mensal de R$150K, CPA de R$80,
ticket medio de R$120. Nao temos upsell nem email pos-compra. Onde esta o
gargalo no meu CVO?
```

### Arquitetura de Lead Magnet
```
Sou consultora de RH para PMEs. Quero criar um lead magnet que converta bem.
Meu publico sao donos de empresas de 20-100 funcionarios com alta rotatividade.
Como voce estruturaria isso dentro do CVO?
```

### Email Lifecycle (ISM)
```
Tenho uma lista de 45K emails mas taxa de abertura caiu de 35% para 12%.
Nao temos segmentacao nem sequencia de win-back. Qual fase do ISM esta
quebrada e como consertar?
```

### Before & After Grid
```
Estou lancando um curso de producao musical para iniciantes. Me ajude a
construir o Before & After Grid completo com as 5 dimensoes.
```

### Value Ladder
```
Tenho um SaaS de gestao financeira para freelancers. Preco unico de R$49/mes.
Como voce montaria a Value Ladder completa, do lead magnet ao profit maximizer?
```

## Exemplos de Prompts — Business Scaling

### Diagnostico de Dependencia do Fundador
```
Faturamos R$5M/ano mas eu trabalho 60h por semana e a empresa para quando
viajo. Tenho 15 funcionarios mas nenhum processo documentado. Por onde comeco?
```

### Implementacao de Scorecards
```
Quero implementar scorecards no meu negocio de agencia de marketing.
Quais metricas voce colocaria e qual cadencia de review?
```

### Growth Engine vs. Fulfillment Engine
```
Minha empresa de consultoria cresce 40% ao ano mas a satisfacao dos clientes
caiu de 9.2 para 7.1. O que esta acontecendo e como mapear isso?
```

### Playbooks para Power Stages
```
Tenho uma empresa de e-commerce com 25 funcionarios. Quais sao os ~12
playbooks que voce criaria e como identificar os power stages?
```

### Distributable Cash
```
Minha receita subiu 60% esse ano mas estou tirando menos dinheiro do negocio
do que ano passado. Me ajude a diagnosticar o problema.
```

## Troubleshooting

| Problema | Causa Provavel | Solucao |
|----------|----------------|---------|
| Respostas genericas | Pergunta vaga, sem contexto de negocio | Reformule com numeros: receita, LTV, CPA, tamanho da equipe |
| Clone nao reframeia a pergunta | Pergunta ja esta no formato diagnostico | Normal — ele so reframeia quando detecta sintoma vs. causa |
| Respostas muito longas | Pediu walkthrough de framework completo | Normal para CVO/SOS. Peca versao resumida se preferir |
| Clone admite que nao sabe | Perguntou sobre personal branding, design, ou tatica de plataforma | Intencional. Ele defere explicitamente nessas areas |
| Saiu do personagem | Contexto muito longo diluiu o system prompt | Relembre o contexto: "Responda como Ryan Deiss, usando CVO" |
| Respostas tipo "it depends" | Modelo base sobrepondo o prompt | Reformule com cenario especifico para forcar diagnostico |
| Clone cita dados incorretos | Alucinacao do modelo base | Verifique dados especificos (numeros, datas) externamente |

## Cenarios Avancados

### Combinar Modos (Marketing + Scaling)
```
Minha agencia fatura R$3M/ano. Marketing funciona bem (CPA baixo, LTV alto)
mas operacao e caos — eu aprovo tudo, nao tenho playbooks, equipe de 20.
Me ajude a mapear o Growth Engine e o Fulfillment Engine separadamente.
```

### Desafiar um Axioma
```
Voce sempre diz que sistema > talento. Mas e quando o fundador E o produto?
Como podcasters, influencers? O sistema funciona sem a pessoa?
```

### Pedir Comparacao com Contemporaneo
```
Qual a diferenca pratica entre a sua abordagem de CVO e o framework de
ofertas do Alex Hormozi? Quando usar cada um?
```
