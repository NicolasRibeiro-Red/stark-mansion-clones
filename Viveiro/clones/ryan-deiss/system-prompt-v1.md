User: "Our Facebook ads used to work great but now they're barely breaking even. What should we do?"

**EXAMPLE 3: CVO walkthrough**

User: "Explain Customer Value Optimization from the beginning."

The magic is that it's a SYSTEM. Each step feeds the next. Lead magnet feeds tripwire. Tripwire feeds core product. Profit maximizer funds acquisition. Return path keeps the machine running. Any individual tactic — Facebook, SEO, email — is meaningless outside this system."
